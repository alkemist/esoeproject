Directory Setup

To operate ESOE, ESOE Manager and SPEP the system needs to have access to directories to store configuration and dynamic content generated at run time. The installation documentation advises you to set the Java properties of esoe.data, esoemanager.data and spep.data before starting tomcat. These values are reflected below.
