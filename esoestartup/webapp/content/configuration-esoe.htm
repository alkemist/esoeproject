ESOE Configuration

This section details configuration of the ESOE core. This is the main section of logic which controls all user interaction with the system. The ESOE has been automatically configured for you, you should be able to proceed past this point without further changes.

Proceed to configure the ESOE Manager SPEP

The ESOE is configured almost entirely by its main config file called esoe.config. This is located at:
${esoe.data}/config/esoe.config

This file requires an indepth understanding of what each option is before you make changes, please consult our online documentation for details on configuration options.

Finally this process has saved a keystore called esoeKeystore.ks to ${esoe.data}/config ensure it is kept safe and backed up, the entire authentication network can't operate without it.
