Database Setup

Before completing this section you should have preconfigured a database and acceptable user for reading and writing to that database.

You should have also executed the SQL initialization script which relates to your database of choice. This is documented in the install process and scripts can be located at $ESOESTARTUP_HOME/databasegeneration/*

The details below will be used to write data to your database and then by the ESOE and ESOE Manager for their future connections. Please ensure appropriate permissions to carry out these tasks are in place for the provided user. $dataRepositoryForm
